# Life in Dots

Visualize your entire lifespan as a grid of dots. Each dot is a month of your life.

A minimalist memento mori dashboard to track your lifespan, gain perspective on time, and cherish every moment.

## Features

- **Interactive Grid**: Map your years, months, days, and even hours.
- **Customizable**: Set your own birth date and expected lifespan.
- **Theme Support**: Seamlessly switch between light and dark modes.
- **Export**: Save your life grid as an image to keep or share.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Avi161/life-dots.git
   ```

2. Navigate to the project directory:
   ```bash
   cd life-dots
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit the local server URL (usually `http://localhost:5173`).
